# Superior Bicycle Forensic Analysis

## Case Overview
This forensic analysis investigates a Windows 8 hard drive image (`InCh05.img`) belonging to Superior Bicycle employee Denise Robinson. The primary objectives were to identify any email accounts using `@outlook.com` associated with Denise Robinson, locate any password files, and document these findings.

## Tools Used
- **Autopsy**: Forensic analysis and registry searches
- **GitHub & GitHub Pages**: Reporting and publishing findings

## Methodology
1. **Registry Search**:
   - Searched for emails with the domain `@outlook.com` and names `Denise` or `Robinson`.
   - Tagged relevant findings and included them in the report.

2. **Password File Search**:
   - Conducted a keyword search for potential password files using terms like `password` and `credentials`.
   - Located files of interest and added them to the report.

## Findings
- **Emails**: Emails associated with `@outlook.com` and the names `Denise` or `Robinson` were identified and documented.
- **Password Files**: Password files found in system directories, tagged, and added to the case report.
